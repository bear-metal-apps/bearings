# Pawfinder

Pawfinder is Bear Metal's FRC scouting app. In Bearings it sits beside Beariscope and the shared packages so scouting data models, services, and UI pieces can move together.

## About

Pawinder is built using flutter. This enables scouting on android while keeping it in the app ecosystem.

## Working on Pawfinder

From the monorepo root:

1. Run `dart pub get`
2. Run `melos run generate` if generated files are stale
3. Start the app with `cd apps/pawfinder && flutter run` or by using melos's IntelliJ run configs
